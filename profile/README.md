## 🌱 About Us

DdanTime은 “딴짓도 의미 있다”는 철학에서 시작한 자기관리 서비스입니다.  
기록이 어렵게 느껴졌던 사람들을 위해, 가볍게, 따뜻하게,  
그리고 칭찬과 위로가 함께하는 공간을 만들고 있습니다.
